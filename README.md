## 👥 프로젝트 팀 구성
- **총 인원:** 4명 (프론트엔드 2명, 백엔드 2명)

## 🎯 프로젝트 개요
**프로젝트 명:** TEAM TAG

### 📌 프로젝트 목적
씨애랑 학술 동아리 태그팀의 홍보 및 활동 내용 공유를 위한 웹사이트 제작

## 📅 개발 기간

## 🌈 주요 기능

### 1. 메인 페이지
- 스크롤 연계 인터랙티브 요소
- 태그팀 소개 및 활동 홍보

### 2. 공지사항
- 공지사항 조회, 검색
- 관리자 전용 공지사항 작성/수정/삭제
- 페이지네이션 기능

### 3. 로드맵
- 프론트엔드/백엔드 학습 로드맵
- 상세 로드맵 페이지

### 4. 작품 소개
- 소프트웨어 작품 전시
- 작품 상세 모달 기능

### 5. 일정 관리
- 캘린더 기반 동아리 일정 확인

### 6. FAQ
- 아코디언 메뉴 방식의 질문/답변

- 팀 최초의 공동 웹사이트 프로젝트 성공적 완료
- 깃 협업, 서버 연결 등 기술적 도전 극복

## 💡 배운 점
- 팀 협업 능력 향상
- 깃허브 협업 워크플로우 학습
- 새로운 기술 스택 습득
- 문제 해결 능력 개선

## 🚀 느낀 점
협업의 가치와 팀워크의 중요성을 깨달은 의미 있는 프로젝트였습니다. 기술적 도전과 서로의 성장을 경험할 수 있었습니다.
